Today I learned about Bash [brace expansion](https://www.gnu.org/software/bash/manual/html_node/Brace-Expansion.html), a convenient way to generate multiple command arguments without repeating yourself.

It's especially handy when creating sibling directories:

```bash
mkdir /some/path/{a,b,c}
```

Which expands to:

```bash
mkdir /some/path/a /some/path/b /some/path/c
```

It also works nicely when renaming something without retyping the directory:

```bash
mv /some/path/{c,d}
```

Which expands to:

```bash
mv /some/path/c /some/path/d
```
