Today I learned that you can upgrade your local golang version using `go env`. You can do this by writing to go's environment key=value store like so: `go env -w KEY=VALUE`.

```bash
go env -w GOTOOLCHAIN=go1.25.4+auto
# then run to apply
go version
go: downloading go1.25.4 (darwin/amd64)
```
